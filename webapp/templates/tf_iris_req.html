Demo #3 - Tensorflow Classification Model (1/2)


    This app demonstrates how to use a Tensorflow model.
    
    It uses the model built by 'iris.ipynb'
    - location: /tensorflow/iris.ipynb
    
    The model is 'tensorflow-iris.h5'
    - location: /model/tensorflow-iris.h5

    The process follows the steps below:
    1. Put the 4 characteristic values to identify the kind of iris.
    2. Send the request to the linear regression model.
        Click 'predict!'' button
    3. Forward the result page to show the prediction from the model.
