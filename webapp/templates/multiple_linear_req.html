Demo #2 - Multiple Linear Regression Model (1/2)


        This app demonstrates how to use a multiple linear regression model.
        
        It uses the model built by 'multiple_linear_regression.ipynb'
        - location: /regression/continuous/linear/multiple_linear_regression.ipynb
        
        The model is 'multiple_linear_regression_model.pkl'
        - location: /data/multiple_linear_regression_model.pkl

        The process follows the steps below:
        1. Select the level of tv promotion budget (High, Medium or Low).
        2. Put the amount of radio promotion budget.
        2. Send the request to the linear regression model.
           Click 'predict!'' button
        3. Forward the result page to show the prediction from the model.


        Equation of the model:
        --------------------------------------
        sales($) = -152.0897 * tv_low($)    +
                    -73.4835 * tv_medium($) + 
                      2.8864 * radio($)     + 
                    217.6367   (default: tv_high)
        --------------------------------------
